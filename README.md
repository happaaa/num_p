# longnum
Arbitrary precision arithmetic package for Dart

Work in progress. Most functions will be on the master branch.


It supports:
-   Addition
-   Subtraction
-   Multiplication
-   Division
-   Modulo
-   Powers
-   Square root
-   Exponential function exp(x)
-   Natural logarithm
-   Comparison operators (>, <=, ==)
-   Maximum and minimum
-   Absolute value
-   Trig functions
